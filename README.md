CurrencyRate WebSite:
Web application, which allows User to check currency rate history and rate fluctuation. In addition User can add new currency rate, which will be immediately diaplayed in chart.


For using this product:
1. Clone this repository to your computer: git clone https://github.com/StarLooorD/SigmaTestTask.git
2. Install requirements: pip install -r requirements.txt
3. Start local server(you can choose any port): python manage.py runserver 5000
